Comandi di GIT

CREATE

Creare un repository locale

& git init

Clonare un repository remoto

& git clone path/di/origin

Modifiche locali

Verificare cosa è successo dall'ultimo commit:

& git status

Aggiungere il file indicato, modificato rispetto all'ultimo commit, nello STAGE:

& git add index.html

Aggiungere tutti i file modificati nello STAGE:

& git add .

Aggiungere tutti i file, o un singolo nello STAGE confermando ogni singola modifica:

& git add -p

Rimuovere il file, precedentemente aggiunto, dallo STAGE, ma mantenendo le modifiche:

& git reset -- index.html

Committare i file, salvare uno snapshot dello stage:

& git commit -m "Messaggio del commit"

Se si è dimenticato qualcosa dall'ultimo commit, combinare i file nello stage con l'ultimo commit (creando un altro commit distinto):

& git commit --amend
